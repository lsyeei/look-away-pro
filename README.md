# Look Away Pro

[English Version](./README-EN.md) | [中文版](./README.md)

一个基于 Qt C++ 开发的桌面护眼提醒应用程序，帮助用户预防长时间使用电脑导致的眼部疲劳。

## 📖 项目简介

Look Away Pro 是一款智能护眼提醒工具，通过定时提醒用户远眺休息，有效缓解眼部疲劳，预防近视和干眼症。程序运行时最小化到系统托盘，不干扰日常工作。

## ✨ 核心功能

### 🎯 智能定时提醒
- **20-20-20 法则**：默认每 20 分钟提醒用户远眺 20 英尺外 20 秒
- **自定义时长**：用户可自由设置工作时间和休息时长
- **强制休息模式**：可选强制模式，休息时间内无法关闭提示画面

### ⏱️ 工具功能
- **计时器**：精确计时显示（HH:MM:SS 格式）
  - 播放/暂停/停止功能
  - 可拖拽的透明窗口
  - 无边框设计，始终置顶
  - 自定义字体、大小、颜色

- **倒计时**：自定义时长的倒计时提醒
  - 支持小时、分钟、秒的灵活设置
  - 完成时闪烁提醒和播放提示音
  - 可拖拽的透明窗口
  - 无边框设计，始终置顶
  - 自定义字体、大小、颜色及提示音

### 🖼️ 个性化提示画面
- **全屏提示**：弹出全屏画面，确保用户注意到休息提醒
- **自定义提示词**：支持自定义文本内容，可使用 `{time}` 和 `{break}` 占位符
- **多样背景选择**：
  - 默认黑色背景
  - 自定义纯色背景
  - 自定义图片背景（支持 BMP、JPG、PNG、GIF）
  - 图片绘制模式：填充、适应、拉伸、平铺
- **字体定制**：可调整提示文字的字体和字号

### 📊 数据统计分析
- **多维度统计**：提供今天、本周、本月的时间统计
- **时间范围查询**：支持自定义时间段查询（默认显示一年记录）
- **本地数据存储**：使用 SQLite 数据库存储，确保数据不丢失
- **界面尺寸**：统计窗口默认为屏幕 1/4 大小

### ⚙️ 灵活配置选项
配置页面分为四大标签页：

**基本配置**：
- 工作时间设置（分钟）
- 休息时间设置（秒）
- 开机自启动选项
- 强制休息选项
- 屏保/锁屏行为：停止计时或重新计时

**提示画面配置**：
- 自定义提示词编辑
- 背景类型选择：图片或纯色
- 图片预览缩略图
- 图片绘制方式选择
- 提示音设置（支持 WAV、MP3、OGG）
- 字体和字号设置
- 实时预览功能（ESC 键退出预览）

**工具配置**：
- 计时器外观设置
  - 字体、大小、颜色配置
- 倒计时外观设置
  - 字体、大小、颜色配置
  - 提示音文件选择

**关于**：
- 程序版本信息
- 作者信息

### 💡 便捷操作
- **系统托盘集成**：程序启动后自动最小化到系统托盘
- **右键菜单**：
  - 🛠️ 工具
    - ⏱️ 计时器：打开计时器窗口
    - ⏳ 倒计时：打开倒计时输入对话框
  - 📈 统计：查看工作和休息统计数据
  - ⚙️ 选项：打开配置页面
  - ⏯️ 暂停/继续：暂停或恢复计时
  - ℹ️ 关于：显示程序信息
  - 🚪 退出：安全退出程序
- **智能暂停**：运行屏保或系统锁屏时自动暂停计时
- **状态同步**：菜单项图标和文字根据状态动态更新

## 🛠️ 技术实现

### 开发环境
- **语言**：C++17
- **框架**：Qt 6.8.3
- **构建工具**：qmake
- **数据库**：SQLite3
- **平台**：Windows、Linux、macOS

### 核心组件
- **ConfigManager**：单例模式的配置管理器，负责 XML 配置文件的读写
- **TimerManager**：计时器管理，精确控制工作和休息时间
- **ScreenStateMonitor**：跨平台屏幕状态监听（Windows WTSAPI/Linux DBus/macOS Cocoa）
- **DatabaseManager**：SQLite 数据库操作，记录统计信息
- **ReminderDialog**：全屏提醒对话框，支持多种背景模式
- **SystemTrayIcon**：系统托盘集成，提供完整右键菜单
- **TimerWindow**：计时器窗口，支持暂停/继续/停止
- **CountdownWindow**：倒计时窗口，支持闪烁提醒和提示音
- **CountdownInputDialog**：倒计时时间输入对话框

### 配置存储
- 配置文件：`config.xml`（存储在应用程序目录）
- 数据库：`LookAwayPro.db`（SQLite 格式）
- 支持开机自启动设置（Windows 注册表）

## 📦 构建与安装

### 前置要求
- Qt 6.x 开发环境
- C++17 兼容编译器
- Windows: Visual Studio 2019+ 或 MinGW
- Linux: GCC 7+ 或 Clang
- macOS: Xcode 10+

### 编译步骤
```bash
# 克隆项目
git clone https://github.com/lsyeei/look-away-pro.git
cd look-away-pro

# 生成 Makefile
qmake LookAwayPro.pro

# 编译
make  # Linux/macOS
# 或
nmake  # Windows MSVC
# 或
mingw32-make  # Windows MinGW

# 运行
./release/LookAwayPro
```

### 安装说明
Windows 用户可直接运行编译后的 `.exe` 文件，程序会自动创建必要的配置文件和数据库。
